# Ha Linh Tran — Portfolio Website

A responsive static portfolio designed for GitHub Pages.

## Files

- `index.html` — website content
- `styles.css` — layout and visual styling
- `script.js` — small utility script
- `assets/ha-linh-headshot.jpeg` — profile image
- `assets/Ha_Linh_Tran_CV.pdf` — downloadable CV

## Publish with GitHub Pages

1. Create a new public repository, for example `portfolio`.
2. Upload all files and folders from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. GitHub will publish the site at:
   `https://YOUR-USERNAME.github.io/portfolio/`

## Customisation

Edit contact details, project summaries and links directly in `index.html`.
The site uses no paid libraries, frameworks or external fonts.
